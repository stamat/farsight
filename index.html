Animate CSS Farsight Animations

Much like WOWJS this script will enable you to activate Animate.css or custom css animations while you scroll. But unlike WOWJS you are able to create parallax effects, reading through paragraphs percentage information and many more by utilizing two powerful classes that expand the expected functionality.
